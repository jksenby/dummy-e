# Devopsizer

### Шаги для запуска проекта
1) Склонируйте себе этот проект  
```sh
git clone https://github.com/dosops/devopsizer-aio
```
2) Установить Docker, docker-compose
3) Пройдите в клонированную папку и запустите проект
```
cd front
npm run build
cd ..
docker-compose up
```

4) Пройдите по ссылке http://locahost:8080
